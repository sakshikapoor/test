# What is this?

Get perfect shadows for the non-designer

# Installation

`npm i test --save `

```
import { testFunction } from 'test';

testFunction ({
    shadow_type: 'soft',
    padding: false
});
```

# Options

TestFunction supports 2 options:

* *shadow_type* - _hard | soft_ (Defaults to soft)
* *padding* - _boolean_ (Deafults to false)
